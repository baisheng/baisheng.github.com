  1. 轻型目录访问协议(LDAP) Lightweight Directory Access Protocol 一种Internet目录访问的服务
  2. WSDL是Web服务描述语言（Web Services Description Language)
  3. CMS是内容管理系统(Contents Management System)
  4. 活动目录访问接口(Active Directory Service Interfaces ，ADSI )
  5. 因特网邮件传送代理(英文: Mail Transfer Agent, 简写为MTA)
  6. 简单报文传送代理协议 SMTP传输邮件(Simple Mail Transfer Protocol)
  7. API (Application Programming Interface) 的意思是:应用编程接口
  8. GTK(GIMP Toolkit)是一个图形用户编程的接口工具
  9. COM是组件式对象模型(Component Object Model)的英文缩写，是组件之间相互接口的规范
  10. DCOM(Distribute COM，分布式构件对象模型)
